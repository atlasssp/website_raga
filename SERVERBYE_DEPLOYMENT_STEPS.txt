DEPLOYMENT METHOD 2: FTP Upload
================================================================================

STEP 1: Get FTP Credentials
- From ServerByte control panel
- Look for "FTP Accounts" section
- Note: hostname, username, password

STEP 2: Download FileZilla (if needed)
- Free FTP client: https://filezilla-project.org/

STEP 3: Connect to Your Server
- Host: ftp.ragabymallika.in (or provided hostname)
- Username: your FTP username
- Password: your FTP password
- Port: 21

STEP 4: Upload Files
- In FileZilla, navigate to "public_html" on the server (right side)
- On your computer (left side), find the "dist" folder
- Drag ALL contents from "dist" folder to "public_html"

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: 404 Error when refreshing pages
SOLUTION: Make sure .htaccess file is uploaded to public_html

PROBLEM: Website shows blank white screen
SOLUTION:
- Press F12 in browser to see errors
- Check if all files uploaded correctly
- Verify Supabase credentials in your .env file

PROBLEM: HTTPS not working
SOLUTION:
- Wait longer (SSL can take up to 24 hours)
- Contact ServerByte support
- Check SSL/TLS Status in cPanel

PROBLEM: Images not loading
SOLUTION:
- Verify "assets" folder uploaded completely
- Check file permissions (should be 644 for files, 755 for folders)
